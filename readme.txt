This plugin archives your Lazyest Gallery to a zip file.

== Description ==

Backup plugin for [Lazyest Gallery](http://wordpress.org/extend/plugins/lazyest-gallery/)

= Features =
* This plugin archives your Lazyest Gallery to gallery-backup.zip
* Backup the full Lazyest Gallery (except thumbnails and slides) or backup captions only
* Download the archive.
* Upload zip files and restore to your Lazyest Gallery

Read more about Lazyest Backup 0.2.0 in [the release post](http://lazye.st?p=4242)


== Installation ==

1. Install using the WordPress plugin installer
2. Activate when you need to backup or restore
